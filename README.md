# 🍫 Chocolate Sales Analytics
### Power BI | DAX | Bookmarks | Field Parameters

## Project Overview
An end-to-end Power BI portfolio project covering data modelling,
DAX measures, interactive dashboard design, and advanced 
interactivity features including bookmarks and field parameters.

## Tools Used
- **Power BI Desktop** — dashboard design & interactivity
- **DAX** — calculated measures & time intelligence
- **GitHub** — project hosting & sharing

## Dataset
- Domain: Chocolate sales (shipments, salesperson, product, geography)
- Tables: Shipments, Salesperson, Product, Geography, Calendar (star schema)

## Key Features Built
- ⭐ Star schema data model (5 tables)
- 📊 DAX measures — Total Sales, Profit %, Low Box Shipments,
  Month-on-Month change, Latest Month values
- 🎯 KPI cards with conditional formatting & reference labels
- 📈 Dynamic trend line using Field Parameters
- 📦 Shipment distribution histogram with zoom slider
- 🗺️ Slicers for Category and Geography filtering
